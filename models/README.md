# Model Directory

Place your trained model file `tomato_model.h5` in this directory.

The model should be a Keras-based deep learning model trained to classify tomato leaf diseases into the following categories:
- Bacterial_spot
- Early_blight
- Late_blight
- Leaf_Mold
- Septoria_leaf_spot
- Spider_mites
- Target_Spot
- Yellow_Leaf_Curl_Virus
- Mosaic_virus
- Healthy

The model should expect input images of size 224x224 pixels with 3 color channels (RGB).
